## Course Information
- Course: Operating Systems Security (CCY4304)
- Lecturer: Prof. Dr. Ayman Adel
- TA: Abdelrahman Solyman

## Project Description
Extended xv6 kernel with three security layers:
1. Phase 1: Role-based user authentication
2. Phase 2: UNIX-style file permissions
3. Phase 3: Syscall audit log
4. Bonus: Automated compliance test suite (17 tests)
